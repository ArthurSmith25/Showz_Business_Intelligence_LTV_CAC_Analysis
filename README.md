# Showz_Business_Intelligence_LTV_CAC_Analysis
Optimización de rentabilidad publicitaria mediante identificación de canales clave. Modelado de LTV, CAC y ROMI con Python. Ejecución de análisis de cohortes y planificación estratégica para la reasignación de presupuesto, logrando una identificación de retornos superiores al 86% en fuentes prioritarias.
